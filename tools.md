# Tools you'll use

| Tool | What it's for |
|:-----|:--------------|
| **Microsoft 365 Copilot Chat** | Ask questions, summarize, draft, brainstorm |
| **Copilot in Word** | Long-form drafting (briefs, press releases, blog posts) |
| **Copilot in PowerPoint** | Pitch decks and campaign reviews |
| **Copilot in Excel** | Campaign performance analysis |
| **Copilot in Outlook** | Email drafting and meeting prep |
| **Copilot Studio – Agent Builder** | Build your own marketing agents |

## Licenses required

- Microsoft 365 Copilot license (per user)
- Access to Copilot Studio for agent-building sessions

{: .note }
> If you're not sure what you have, ask your IT admin or check the **My Account** page in Microsoft 365.
