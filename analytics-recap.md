# Analytics Recap

[COPILOT IN EXCEL]{: .label .label-blue } [POWERPOINT]{: .label .label-purple } [45 MINUTES]{: .label .label-green }

Take last quarter's campaign performance and produce a 5-slide readout your CMO will actually read.

## The prompt (Excel)

{: .note }
> *"Summarize this campaign performance data. Highlight the three best-performing channels by ROI, the three worst, and one surprising insight. Output as a markdown table."*

## The prompt (PowerPoint)

{: .note }
> *"Create a 5-slide deck from this summary: title slide, top performers, underperformers, surprising insight, recommendations for next quarter."*
