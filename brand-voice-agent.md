# Core 3 · Build a Brand Voice Agent

[COPILOT STUDIO]{: .label .label-blue } [AGENT BUILDER]{: .label .label-purple } [90 MINUTES]{: .label .label-green }

The capstone session. By the end of this hour, you'll have a published agent that anyone in Marketing can use to check copy against your brand voice.

## What you'll build

A **Brand Voice Agent** that:
- Reads any draft (email, social post, web copy)
- Flags off-brand phrases
- Suggests on-brand rewrites
- Cites the section of the brand guide it referenced

## Steps

### 1. Open Copilot Studio
Go to [copilotstudio.microsoft.com](https://copilotstudio.microsoft.com) and choose **Create > Agent**.

### 2. Describe the agent
Paste this in the description box:

{: .note }
> *"You are Contoso's Brand Voice Agent. When given a piece of marketing copy, you check it against the attached brand voice guide. You flag any phrase that breaks our voice rules (no jargon, no superlatives, no exclamation marks) and suggest a rewrite. Always cite the section of the brand guide you used."*

### 3. Add knowledge
Upload the **brand voice guide** and **3 approved campaign emails** as knowledge sources.

### 4. Test it
Paste a draft email into the test panel. Verify the agent:
- Catches at least one off-brand phrase
- Suggests a fix
- Cites the brand guide

### 5. Publish
Share with the Marketing team via the Microsoft 365 Copilot **agent gallery**.

## Success criteria

- Agent published to your team
- At least 3 colleagues have tried it
- One piece of copy improved before going live
