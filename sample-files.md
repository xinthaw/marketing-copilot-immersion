## How to add or replace a file

1. In your repo, click into the **`assets/`** folder
2. Click **Add file → Upload files**
3. Drag the file in, then **Commit changes**
4. The file is immediately available at `{{ site.baseurl }}/assets/your-file.ext`

## File naming tips

- Lowercase only
- Use `-` instead of spaces (`brand-voice-guide.pdf`, not `Brand Voice Guide.pdf`)
- Keep extensions standard: `.pdf`, `.docx`, `.xlsx`, `.pptx`
